\subsection{ベクトルの和}

2つの$n$次元ベクトル$\bs{v},\bs{w}$に対して，
その\textbf{和}$\bs{v}+\bs{w}$を次のように定義する．
$\bs{v}+\bs{w}$が数の組$(v_1,	\cdots	,	v_n)$と$(w_1	,	\cdots	,	w_n)$でそれぞれ表わされる
ときに，$\bs{v}+\bs{w}$は$(v_1	+	w_1	,	\cdots	v_n	+	w_n)$で表わされる．

この規則を矢印としてのベクトルの場合に言い換えると次のようになる．

$\bs{v}$が始点$p$，終点$q$の矢印で表わされるとする．このとき$\bs{w}$を
表わす矢印をとり，$q$が始点となるように平衡移動し，その終点を$r$とおく．
$p$を始点とし$r$を終点とする矢印の表わすベクトルが$\bs{v}+\bs{w}$である（図\ref{fig1x2}).
\begin{figure}[hbtp]	
		\centering	
		\includegraphics[bb=0 0 800 600,clip,width=5cm,height=4cm]{fig1x2.bmp}	
		\caption{複素数の表示}	
		\label{fig1x2}	
\end{figure}	
あるいは，$\bs{v}$と$\bs{w}$を同じ点$p$を始点として描き，その終点を$q$および$r$とする．
$p,q,r$を頂点とする平行4辺形の第4の頂点を$o$とすると，$p$を始点,$o$を終点とする
矢印が表わすベクトルは$\bs{v}+\bs{w}$である(図\ref{fig1x3}).
\begin{figure}[hbtp]	
		\centering	
		\includegraphics[bb=0 0 800 600,clip,width=4cm,height=3cm]{fig1x3.bmp}	
		\caption{平行四辺形の法則}	
		\label{fig1x3}	
\end{figure}	
このように和を定めたとき，次のことが成り立つ．
\begin{list}{}{}
	\item	(ⅰ) $\bs{u}	+	(\bs{v}	+	\bs{w})	=	(	\bs{u}	+	\bs{v})	+	\bs{w}$
							（結合法則）．
	\item	(ⅱ) $\bs{0}$なるベクトルが存在して，$\bs{0}	+	\bs{v}	=	\bs{v}+	\bs{0}	=	\bs{v}$が任意のベクトル$\bs{v}$に対して成立する（単位元の存在）．
	\item	(ⅲ) 任意のベクトル$\bs{v}$に対して$-\bs{v}$が存在して$-\bs{v}	+	\bs{v}	=	\bs{v}	+	-\bs{v}
							=	\bs{0}$が成り立つ（逆元の存在)．
	\item	(ⅳ) $\bs{v}+\bs{w}	=	\bs{w}	+	\bs{v}$（交換法則)
\end{list}
集合とその間の演算が(ⅰ),(ⅱ),(ⅲ)を満たすとき\textbf{群}と呼び，(ⅳ)も成立するとき
\textbf{可換(アーベル)群}と呼ぶ．

さらにベクトルには実数を描けることができる．すなわち$\bs{v}$を$(v_1,	\cdots	,	v_n)$で
表わされるベクトル,$C$を実数とするとき，$C\bs{v}	=	(Cv_1,	\cdots	,	Cv_n)$と定義する．
矢印で表わしたときは，$C\bs{v}$は$\bs{v}$と同じ方向に向いた，長さが$\bs{v}$の$C$倍の
矢印が表わすベクトルであるとする．（$C$が負のときは，$C\bs{v}$は$\bs{v}$と反対の
方向を向いた，長さが$\bs{v}$の$|C|$倍の矢印が表わすベクトルであると定める.)
これに対して次の規則が成り立つ．
\begin{list}{}{}
	\item	(ⅴ) $C	(\bs{v}+\bs{w})=C\bs{v}+C\bs{w}.$
	\item	(ⅵ) $-1\bs{v}	=	-\bs{v},	\quad	0	\bs{v}=\bs{0}.$
\end{list}
集合上に足し算および実数を掛ける操作が定義され，(ⅰ)-(ⅵ)の規則が成り立つとき，
これはベクトル空間となる．
